# Week 3: Advanced AI Techniques

> **Goal:** Master multimodal AI, local model deployment, fine-tuning concepts, advanced prompting, and AI evaluation.

## 🎯 Learning Objectives

By the end of this week, you will:
- Send images to AI models for visual analysis (Vision)
- Run AI models locally with Ollama (zero cost, full privacy)
- Understand when to use RAG vs fine-tuning vs prompt engineering
- Master advanced prompting patterns: ReAct, Tree-of-Thought, Self-Consistency
- Build automated AI evaluation harnesses with LLM-as-Judge
